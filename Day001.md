## 1.Write a program that takes three numbers and prints their sum. Every number is given on a separate line

##### SOURCE CODE:
```python
a = int(input())
b = int(input())
c = int(input())
print(a+b+c)
```
**Example input**

```python
2 
8 
6
```

**Example output:**

```python
16
```
